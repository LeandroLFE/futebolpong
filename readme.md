# Install

## # Create Python virtual envinronment
```
    python -m venv venv
```
## # Activate virtual envinronment
```
    .\venv\Scripts\activate
```
## # Install requirements
```
    pip install -r requirements.txt
```
## # Run
```
    python pong.py
```
